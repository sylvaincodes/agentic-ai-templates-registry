# 🧠 Idea to Roadmap Skill

## What it does
Turns vague ideas into structured execution plans with clear roadmaps.

---

## When to use it
- Startup ideas
- SaaS concepts
- Product brainstorming
- Feature planning

---

## Example input
> "I want to build a tool for developers to manage tasks better"

---

## What it produces
- Clarified problem definition
- Refined product idea
- MVP scope
- Step-by-step roadmap
- Risks and assumptions

---

## Philosophy
This skill is strict about execution. It does NOT encourage vague brainstorming — it forces clarity and action.
